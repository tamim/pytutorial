Tutorial 9: Data Structures Revisited - Combining Lists, Tuples, Sets & Dictionaries 🔄📊

1️⃣ Why Combine Data Structures?

Real-world programming problems often require the use of multiple data structures together. By combining lists, tuples, sets, and dictionaries, you can:

✅ Store and process complex data efficiently. ✅ Perform operations like filtering, sorting, and grouping. ✅ Use visualization tools to represent data better.

🔍 In this tutorial, you’ll: - Work on examples that combine multiple data structures. - Learn how to process and visualize data. - Solve hands-on exercises that reflect real-world problems.

2️⃣ Example: Organizing Student Data 🎓

Imagine you are building a system to track students' grades and attendance.


students = {
    "Alice": {"grades": [85, 90, 78], "attendance": {"present": 18, "absent": 2}},
    "Bob": {"grades": [92, 88, 94], "attendance": {"present": 20, "absent": 0}},
}
Calculate average grades

for name, info in students.items():
    avg_grade = sum(info["grades"]) / len(info["grades"])
    print(f"{name}'s average grade: {avg_grade}")
✅ Try-it: Modify the program to find the student with the highest average grade.

3️⃣ Using Sets and Lists Together 🔢

Let’s say we are tracking students who attended a math workshop.


math_workshop = {"Alice", "Charlie", "David"}
science_workshop = {"Bob", "Charlie", "Eve"}
both_workshops = math_workshop & science_workshop  # Intersection
only_math = math_workshop - science_workshop  # Difference
all_students = math_workshop | science_workshop  # Union
print("Students in both workshops:", both_workshops)
print("Students only in math workshop:", only_math)
print("All students who attended at least one workshop:", all_students)
✅ Mini Challenge: Create a program that tracks club memberships using sets.

4️⃣ Visualizing Data with Matplotlib 📊

Let’s use Matplotlib to visualize student attendance.

Install Matplotlib (if not installed)

`sh pip install matplotlib

Plotting Student Attendance


import matplotlib.pyplot as plt
students = ["Alice", "Bob", "Charlie"]
attendance = [18, 20, 15]  # Number of days present
plt.bar(students, attendance, color=['blue', 'green', 'orange'])
plt.xlabel("Students")
plt.ylabel("Attendance")
plt.title("Student Attendance Record")
plt.show()

5️⃣ Additional Examples 📌

Example 1: Social Media Friend Tracking 🌐

Track friendships using dictionaries and sets. 

friends = {
    "Alice": {"Bob", "Charlie"},
    "Bob": {"Alice", "David"},
    "Charlie": {"Alice", "Eve"}
}
Find mutual friends

def mutual_friends(user1, user2):
    return friends[user1] & friends[user2]
print(mutual_friends("Alice", "Bob"))  # Output: {'Charlie'}
✅ Challenge: Modify the program to add and remove friends dynamically.

#### Alternative: Using Lists Instead of Sets 


friends_list = {
    "Alice": ["Bob", "Charlie"],
    "Bob": ["Alice", "David"],
    "Charlie": ["Alice", "Eve"]
}
✅ Pros & Cons: - Sets ensure uniqueness, making lookups faster. - Lists maintain order but allow duplicates, making searches slower.

Example 2: Weather Tracking System ☁️

Use tuples as dictionary keys to store weather data. 

weather = {
    ("New York", "2024-05-01"): {"temperature": 72, "humidity": 60},
    ("Los Angeles", "2024-05-01"): {"temperature": 80, "humidity": 50},
}
Access data

print(weather[("New York", "2024-05-01")]["temperature"])  # Output: 72
✅ Challenge: Expand the system to track wind speed as well.

Example 3: Library Management System 📚

Manage book loans using dictionaries and sets. 

library = {
    "The Hobbit": {"available": 3, "borrowed": set()},
    "1984": {"available": 2, "borrowed": {"Alice"}},
