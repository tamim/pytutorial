Tutorial 13: Mini-Projects & Advanced Challenges 🚀

1️⃣ Why Build Projects? 🏗️

Building projects helps reinforce programming concepts by applying them to real-world scenarios. This chapter contains mini-projects and challenges to improve your problem-solving, coding efficiency, and creativity.

✅ What You’ll Gain from This Chapter:

- Hands-on experience with Python. - Confidence in writing complete programs. - Understanding how different programming concepts work together.

Each project is categorized into three difficulty levels:

  1. Small-Scale Mini-Projects (Foundation Level) → Reinforce basic concepts.
  1. Medium-Scale Projects (Problem-Solving Focus) → Apply multiple concepts.
  1. Advanced Challenges (Critical Thinking & Optimization) → Work on efficiency and scalability.

2️⃣ Small-Scale Mini-Projects (Foundation Level) 🏗️

These projects reinforce basic concepts like loops, conditionals, functions, and file handling.

Project 1: Simple Quiz Game 🎯

📌 Concepts Used: Functions, loops, conditionals, user input.

🔹 Problem Statement: Create a Python program that asks the user 5 multiple-choice questions, tracks the score, and displays the final result.

🔹 Hints:

- Store questions and answers in a dictionary. - Use a loop to iterate through the questions. - Keep track of the correct answers.

🔹 Try-it: Modify the game to select random questions each time.

Project 2: Text Formatter 📝

📌 Concepts Used: String manipulation, file handling.

🔹 Problem Statement: Write a program that reads a text file, removes extra spaces, converts it to lowercase, and saves the formatted text in a new file.

🔹 Hints:

- Use open() to read and write files. - Use .strip() to remove extra spaces. - Convert all text to lowercase using .lower().

🔹 Try-it: Extend the project to count the word frequency in the text.

Project 3: To-Do List ✅

📌 Concepts Used: Lists, dictionaries, user input, file handling.

🔹 Problem Statement: Build a simple To-Do List app where users can add, remove, and view tasks.

🔹 Hints:

- Store tasks in a dictionary with unique IDs. - Provide menu options to add, remove, and display tasks. - Save tasks to a file so they persist.

🔹 Try-it: Add a due date feature and sort tasks by date.

3️⃣ Medium-Scale Projects (Problem-Solving Focus) 🔍

These projects combine multiple concepts (data structures, file handling, error handling) to build practical applications.

Project 4: Student Grade Tracker 🎓

📌 Concepts Used: Dictionaries, loops, file handling.

🔹 Problem Statement: Create a program to store student names and grades, calculate averages, and generate a report.

🔹 Hints:

- Store student names as keys and grades as values in a dictionary. - Allow users to add students, update grades, and view reports. - Save data in a file.

🔹 Try-it: Implement a feature to rank students by performance.

Project 5: Expense Tracker 💰

📌 Concepts Used: Lists, dictionaries, file handling.

🔹 Problem Statement: Write a program that lets users log expenses, categorize them, and display spending summaries.

🔹 Hints:

- Store expenses as a list of dictionaries. - Provide options to add an expense, remove an expense, and view total spending. - Categorize expenses and calculate the total spent in each category.

🔹 Try-it: Integrate Matplotlib to visualize spending trends.

Project 6: Simple Encryption Tool 🔐

📌 Concepts Used: String manipulation, loops.

🔹 Problem Statement: Build a Caesar cipher encryption tool where users can encrypt and decrypt messages.

🔹 Hints:

- Shift each letter by a certain number of positions. - Allow users to choose encryption (shift forward) or decryption (shift backward).

🔹 Try-it: Support custom shift values and different cipher methods.

4️⃣ Advanced Challenges (Critical Thinking & Optimization) 🔥

These projects focus on algorithmic thinking & efficiency.

Project 7: Pathfinding in a Grid 🗺️

📌 Concepts Used: 2D lists, loops, recursion.

🔹 Problem Statement: Create a simple maze solver that finds a path in a grid-based maze.

🔹 Hints:

- Represent the maze as a 2D list (0 = open path, 1 = wall). - Use Depth-First Search (DFS) or Breadth-First Search (BFS) to find a path.

🔹 Try-it: Visualize the maze solution using Matplotlib.

Project 8: Library Management System 📚

📌 Concepts Used: Object-Oriented Programming (OOP), dictionaries, file handling.

🔹 Problem Statement: Implement a library system where users can borrow, return, and search for books.

🔹 Hints:

- Use a Book class with attributes like title, author, and availability. - Implement functions to borrow a book, return a book, and check availability. - Store books in a dictionary.

🔹 Try-it: Implement a penalty system for overdue books.

💡 Tip: Keep coding and solving new challenges to sharpen your skills! 🚀
