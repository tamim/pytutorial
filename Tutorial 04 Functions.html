Tutorial 4: Functions - Writing Reusable Code 🔧

1️⃣ What Are Functions? 🏗️

A function is a reusable block of code that performs a specific task. Instead of repeating code, you can call a function whenever needed!

✅ Benefits of Using Functions: - Reusability: Write once, use multiple times. - Readability: Makes code organized and easier to understand. - Maintainability: Easier to modify and fix bugs.

2️⃣ Defining and Calling Functions 📞

Creating a Function

Use def to define a function, followed by a function name and parentheses ().


def greet():
    print("Hello, welcome to Python!")

Calling a Function

To execute a function, type its name followed by ().


greet()  # Output: Hello, welcome to Python!

3️⃣ Functions with Parameters 📥

You can pass values (parameters) to functions to make them more flexible.


def greet(name):
    print(f"Hello, {name}!")
greet("Alice")  # Output: Hello, Alice!

💡 Mini Challenge: Modify greet to take a greeting message as an additional parameter.

4️⃣ Returning Values from Functions 📤

A function can return a value using return.


def add(a, b):
    return a + b
result = add(3, 5)
print(result)  # Output: 8

💡 Mini Challenge: Write a function multiply that takes two numbers and returns their product.

5️⃣ Function Scope: Local vs Global Variables 🌎

Local Scope (Inside Function)

Variables inside a function are local and exist only within that function.


def show_number():
    num = 10  # Local variable
    print(num)
show_number()
print(num)  # ❌ Error: 'num' is not defined outside the function

Global Scope (Outside Function)

A variable declared outside any function is global and can be used anywhere.


message = "Hello"
def greet():
    print(message)  # Accessing global variable
greet()

6️⃣ Default and Keyword Arguments 🛠️

Default Parameters

You can assign default values to parameters.


def greet(name="User"):
    print(f"Hello, {name}!")
greet()      # Output: Hello, User!
greet("Sam") # Output: Hello, Sam!

Keyword Arguments

Use named arguments for better readability.


def describe_pet(animal, name):
    print(f"I have a {animal} named {name}.")
describe_pet(animal="dog", name="Buddy")
describe_pet(name="Whiskers", animal="cat")

💡 Mini Challenge: Modify describe_pet to include a default value for animal as "dog".

7️⃣ Lambda Functions (Short Anonymous Functions) ⚡

A lambda function is a short function that has no name.


square = lambda x: x * x
print(square(5))  # Output: 25
