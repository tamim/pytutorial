Tutorial 2: Control Flow - Making Decisions in Code 🎮

1️⃣ What is Control Flow?

In everyday life, we make decisions: - If it's raining, take an umbrella. - If you have enough money, buy an ice cream, otherwise save up.

In Python, control flow allows our programs to make decisions and repeat actions. We achieve this using if-statements, loops, and logical operators.

2️⃣ If-Else Statements: Making Decisions

An if-statement checks a condition and executes a block of code only if the condition is True.

Basic If-Else Statement

age = int(input("Enter your age: "))
if age >= 18:
    print("You are eligible to vote!")
else:
    print("You are not old enough to vote.")

3️⃣ Comparison & Logical Operators

Comparison Operators

These operators are used to compare values:

| Operator | Meaning | Example | |----------|----------------------|------------| | == | Equal to | x == y | | != | Not equal to | x != y | | > | Greater than | x > y | | < | Less than | x < y | | >= | Greater or equal | x >= y | | <= | Less or equal | x <= y |

Logical Operators

Logical operators combine multiple conditions:

| Operator | Meaning | Example | |----------|-------------|-----------------------| | and | Both must be True | x > 0 and x < 10 | | or | At least one True | x < 0 or x > 10 | | not | Negates condition | not (x > 5) |

Example Using Logical Operators

score = int(input("Enter your test score: "))
if score >= 90 and score <= 100:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

4️⃣ Loops: Repeating Actions 🔄

Loops allow us to repeat code efficiently instead of writing repetitive instructions manually.

For Loop (Repeats a known number of times)

for i in range(1, 6):
    print("Iteration", i)
✅ Output: ` Iteration 1 Iteration 2 Iteration 3 Iteration 4 Iteration 5 `

While Loop (Repeats while a condition is True)

count = 3
while count > 0:
    print("Counting down:", count)
    count -= 1
print("Blast off!")

5️⃣ Controlling Loops: Break & Continue

Break: Stop the Loop Early

for num in range(1, 10):
    if num == 5:
        print("Stopping early!")
        break
    print(num)
✅ Output: ` 1 2 3 4 Stopping early! `

Continue: Skip to the Next Iteration

for num in range(1, 6):
    if num == 3:
        continue
    print(num)
✅ Output: ` 1 2 4 5 `
