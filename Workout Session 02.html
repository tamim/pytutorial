2️⃣ Warm-Up: Quick Examples (30%) 🏁

Example 1: Find the First and Last Character of a String


def first_last(s):
    return s[0] + s[-1] if len(s) > 1 else s * 2
print(first_last("hello"))  # Output: ho
print(first_last("a"))  # Output: aa
✅ Try-it: Modify this function to return the first and last two characters if the string length is more than 4.

Example 2: Summing Digits in a Number


def sum_digits(n):
    return sum(int(digit) for digit in str(n))
print(sum_digits(1234))  # Output: 10
✅ Modify this to return the product of the digits instead.

Example 3: Checking if a String is a Palindrome


def is_palindrome(s):
    return s == s[::-1]
print(is_palindrome("racecar"))  # Output: True
print(is_palindrome("python"))  # Output: False
✅ Modify this function to ignore case and spaces while checking for palindromes.
