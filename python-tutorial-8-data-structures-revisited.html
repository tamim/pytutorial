Tutorial: Combining Data Structures for Problem Solving

Welcome to the final tutorial on data structures! By now, you’ve mastered the basics of lists, tuples, dictionaries, and sets. In this tutorial, we’ll combine these tools to solve practical problems and explore advanced techniques like comprehensions and efficient membership checks.

1. Revisiting the Basics

Before diving into advanced scenarios, let’s recall the strengths of each data structure:

  • Lists: Ordered and mutable collections, ideal for sequences of items.
  • Tuples: Immutable collections, perfect for fixed or constant data.
  • Dictionaries: Key-value pairs for organizing data efficiently.
  • Sets: Unordered collections of unique items, great for removing duplicates and performing mathematical operations.

Example 2: Using Sets with Lists

Track attendees for multiple events using sets to find common or unique participants.

event1 = {"Alice", "Bob", "Charlie"}
event2 = {"Bob", "David", "Alice"}

# Combine attendees
all_attendees = event1 | event2

# Common attendees
common_attendees = event1 & event2

print(all_attendees)  # Output: {'Alice', 'Bob', 'Charlie', 'David'}
print(common_attendees)  # Output: {'Alice', 'Bob'}

Example 3: Tuples as Dictionary Keys

Use tuples as keys for storing multi-dimensional data like weather by city and date.

weather = {
    ("Sydney", "2024-11-01"): 25,
    ("Melbourne", "2024-11-01"): 20,
}

# Access weather data
print(weather[("Sydney", "2024-11-01")])  # Output: 25

3. When to Use Sets Instead of Lists

Membership Check: Faster in Sets

Checking if an item exists in a set is faster than checking in a list, especially for large datasets.

Example: Comparing Membership Check Speed

import timeit

# Create large list and set
large_list = list(range(1_000_000))
large_set = set(large_list)

# Time membership checks
list_time = timeit.timeit('999_999 in large_list', globals=globals(), number=1000)
set_time = timeit.timeit('999_999 in large_set', globals=globals(), number=1000)

print(f"List time: {list_time} seconds")
print(f"Set time: {set_time} seconds")

Output:

List time: 2.3 seconds
Set time: 0.002 seconds

Removing Duplicates from a List

Use a set to clean up duplicates from a list:

names = ["Alice", "Bob", "Alice", "Charlie", "Bob"]
unique_names = list(set(names))
print(unique_names)  # Output: ['Alice', 'Bob', 'Charlie']

4. Reinforcing Functions

What Happens When We Pass Data Structures to a Function?

Python passes all arguments by object reference:

  • Lists, sets, and dictionaries can be modified inside the function.
  • Tuples remain unchanged because they are immutable.

Example:

def modify_structures(lst, tup, s, d):
    lst.append(99)
    s.add("new_item")
    d["new_key"] = "new_value"

my_list = [1, 2, 3]
my_tuple = (1, 2, 3)
my_set = {1, 2, 3}
my_dict = {"key1": "value1"}

modify_structures(my_list, my_tuple, my_set, my_dict)

5. List, Dictionary, and Set Comprehensions

Python comprehensions let you create new data structures efficiently.

List Comprehensions

numbers = [1, 2, 3, 4, 5]
squared = [n ** 2 for n in numbers]

Dictionary Comprehensions

squared_dict = {n: n ** 2 for n in numbers}

Set Comprehensions

unique_squared = {n ** 2 for n in numbers}
