2. Finding Characters: Indexing üìç

Understanding String Indexes

Think of a string like a row of boxes, where each box holds one character:

# The string "Python":
#  P   y   t   h   o   n
# [0] [1] [2] [3] [4] [5]  (counting from front)
# [-6][-5][-4][-3][-2][-1] (counting from back)

word = "Python"

# Get characters from the front (positive indexes)
first = word[0]    # Gets 'P'
second = word[1]   # Gets 'y'
third = word[2]    # Gets 't'

print(f"First character: {first}")
print(f"Second character: {second}")
print(f"Third character: {third}")

# Get characters from the end (negative indexes)
last = word[-1]         # Gets 'n'
second_last = word[-2]  # Gets 'o'

print(f"Last character: {last}")
print(f"Second to last: {second_last}")

Practice with Indexes

# Let's write a program that prints parts of your name
name = input("Enter your name: ")
print("First letter:", name[0])
print("Last letter:", name[-1])

# Be careful! If you try to use an index that doesn't exist,
# Python will give you an error
# For example: name[100] will cause an error if name is shorter than 100 characters

3. Getting Parts of Strings: Slicing ‚úÇÔ∏è

Understanding Slicing

# Slicing format: word[start:end]
# Gets characters from 'start' up to (but not including) 'end'

#  P   y   t   h   o   n
# [0] [1] [2] [3] [4] [5]
# [0:2] gives "Py"    ‚ñà‚ñà‚ñë‚ñë‚ñë‚ñë
# [2:4] gives "th"    ‚ñë‚ñë‚ñà‚ñà‚ñë‚ñë
# [-2:] gives "on"    ‚ñë‚ñë‚ñë‚ñë‚ñà‚ñà
# [:3]  gives "Pyt"   ‚ñà‚ñà‚ñà‚ñë‚ñë‚ñë

word = "Python"

# Get first two characters
first_two = word[0:2]    # Gets 'Py'
print("First two:", first_two)

# Get last two characters
last_two = word[-2:]     # Gets 'on'
print("Last two:", last_two)

# Get middle characters
middle_chars = word[2:4]  # Gets 'th'
print("Middle characters:", middle_chars)

Slicing Shortcuts

word = "Python"

# Skip the start number to begin from start
up_to_three = word[:3]    # Same as word[0:3], gets 'Pyt'
print("Up to third:", up_to_three)

# Skip the end number to go until end
from_three = word[3:]     # Gets 'hon'
print("From third:", from_three)

# Use negative numbers to count from the end
last_three = word[-3:]    # Gets 'hon'
print("Last three:", last_three)

Fun Examples

message = "Hello, World!"

# Get just the hello
hello = message[0:5]
print(hello)    # Shows: Hello

# Get just the world
world = message[7:12]
print(world)    # Shows: World

# Get letters from end
last_few = message[-6:]
print(last_few)    # Shows: World!

4. Useful String Methods üõ†Ô∏è

Making Text Uppercase or Lowercase

message = "Hello Python"

# Make it LOUD
loud = message.upper()
print("Loud:", loud)         # Shows: HELLO PYTHON

# Make it quiet
quiet = message.lower()
print("Quiet:", quiet)       # Shows: hello python

# Try it yourself!
name = input("Enter your name: ")
print("Your name shouting:", name.upper())
print("Your name whispering:", name.lower())

# title() makes first letter of each word uppercase
name = "john smith"
proper_name = name.title()
print(proper_name)    # Shows: John Smith

Cleaning Up Extra Spaces

# Sometimes users type extra spaces by mistake
messy_name = "   Alex   "
clean_name = messy_name.strip()

print(f"Messy: '{messy_name}'")     # Shows: '   Alex   '
print(f"Clean: '{clean_name}'")      # Shows: 'Alex'

# Real-world example
username = input("Enter username: ")
clean_username = username.strip()
print(f"Logged in as: '{clean_username}'")

Counting Letters and Words

sentence = "banana"
a_count = sentence.count('a')
print(f"The letter 'a' appears {a_count} times")    # Shows: 3

# Let's count in a longer text
text = "Mississippi river"
print("'s' appears:", text.count('s'))    # Shows: 4
print("'i' appears:", text.count('i'))    # Shows: 4

Finding Text Position

message = "I love Python programming"
position = message.find("Python")
print(f"'Python' starts at position: {position}")    # Shows: 7

# If the word isn't there, find() returns -1
result = message.find("Java")
print(f"'Java' position: {result}")    # Shows: -1

# Use this to check if a word exists
word_to_find = "Python"
if message.find(word_to_find) != -1:
    print(f"Found {word_to_find}!")
else:
    print(f"Did not find {word_to_find}")

Checking Start and End of Strings

# startswith() checks if string begins with specific text
filename = "python_code.py"
if filename.startswith("python"):
    print("This file starts with python!")    # Shows: This file starts with python!

# endswith() checks if string ends with specific text
if filename.endswith(".py"):
    print("This is a Python file!")          # Shows: This is a Python file!

# Real-world example
files = ["homework.pdf", "notes.txt", "program.py", "document.pdf"]
pdf_files = 0
for file in files:
    if file.endswith(".pdf"):
        pdf_files = pdf_files + 1
print(f"Found {pdf_files} PDF files")    # Shows: Found 2 PDF files

5. ASCII Values - Secret Number Codes! üî°

Understanding ASCII

Every character in your computer has a secret number code:

# Get the secret code (ASCII value)
print("Secret code for 'A':", ord('A'))    # Shows: 65
print("Secret code for 'a':", ord('a'))    # Shows: 97
print("Secret code for '1':", ord('1'))    # Shows: 49

# Turn secret codes back into letters
print("Code 66 gives us:", chr(66))    # Shows: B
print("Code 98 gives us:", chr(98))    # Shows: b

# Exploring more ASCII values
for num in range(65, 71):  # Try first 6 uppercase letters
    print(f"Code {num} is letter {chr(num)}")

    secret = secret.replace('i', '3')
    secret = secret.replace('o', '4')

    return secret

9. Common Mistakes to Avoid ‚ö†Ô∏è

  1. Forgetting that index starts at 0

    name = "Alex"
# first letter is name[0], not name[1]

  2. Using index that doesn't exist

    name = "Bob"
# name[3] will cause error - only have 0,1,2

  3. Mixing up strings and numbers

    age = "15"
# age + 1 will cause error
# need to convert: int(age) + 1

  4. Forgetting to save method results

    text = "Hello"
text.upper()        # This doesn't change text
text = text.upper() # This saves the change
