Tutorial 8: Sets & Dictionaries - Organizing and Accessing Data Efficiently 📚

1️⃣ Why Learn About Sets and Dictionaries?

In Python, sets and dictionaries provide efficient ways to store and retrieve data.

✅ Why Use Sets and Dictionaries? - Sets help store unique values and perform operations like union and intersection. - Dictionaries store key-value pairs, making data lookup fast and efficient.

2️⃣ Sets: A Collection of Unique Elements 🔢

Creating a Set


fruits = {"apple", "banana", "cherry", "apple"}  # Duplicate "apple" removed automatically
print(fruits)  # Output: {'apple', 'banana', 'cherry'}
✅ Use Cases: Removing duplicates, membership checks, set operations.

Adding and Removing Elements


fruits.add("orange")  # Add an element
fruits.remove("banana")  # Remove an element
print(fruits)

Set Operations (Union, Intersection, Difference)


set_a = {1, 2, 3, 4}
set_b = {3, 4, 5, 6}
print(set_a | set_b)  # Union: {1, 2, 3, 4, 5, 6}
print(set_a & set_b)  # Intersection: {3, 4}
print(set_a - set_b)  # Difference: {1, 2}

💡 Mini Challenge: Create two sets and find their union and intersection.

3️⃣ Dictionaries: Key-Value Pairs for Fast Lookups 📖

Creating a Dictionary


student = {"name": "Alice", "age": 22, "major": "Computer Science"}
print(student["name"])  # Output: Alice
✅ Use Cases: Storing structured data, fast lookups, mapping relationships.

Adding and Modifying Values


student["age"] = 23  # Modify value
student["grade"] = "A"  # Add new key-value pair
print(student)

Iterating Over a Dictionary


for key, value in student.items():
    print(f"{key}: {value}")

💡 Mini Challenge: Create a dictionary storing book titles and their authors, then print each pair.

4️⃣ Dictionary Methods & Operations 🛠️


print(student.keys())  # Get all keys
print(student.values())  # Get all values
print("age" in student)  # Check if key exists
student.pop("major")  # Remove a key-value pair
