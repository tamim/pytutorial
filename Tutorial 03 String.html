Tutorial 3: String Adventures - Playing with Text! ✍️

1️⃣ What are Strings? 📜

A string is a sequence of characters enclosed in quotes. You can use either single (') or double (") quotes.


message = "Hello, Python!"
name = 'Alice'

✅ Checkpoint: Try assigning different sentences to a string variable and printing them.

2️⃣ Combining and Formatting Strings

String Concatenation

Use + to join strings together.


first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name
print(full_name)  # Output: John Doe

💡 Try-it: Ask the user for their first and last name and print their full name.

String Formatting (f-strings)

Instead of using +, f-strings provide a cleaner way to format text.


name = "Alice"
age = 25
print(f"My name is {name} and I am {age} years old.")

💡 Mini Challenge: Modify the program to print a sentence using different variables.

3️⃣ String Methods 🛠️

Python provides many built-in methods to work with strings.

Changing Case


text = "Hello, Python!"
print(text.upper())  # HELLO, PYTHON!
print(text.lower())  # hello, python!
print(text.title())  # Hello, Python!

Removing Extra Spaces


user_input = "   Python   "
print(user_input.strip())  # "Python"

Replacing and Finding Text


text = "I love Java!"
print(text.replace("Java", "Python"))  # I love Python!
print(text.find("Java"))  # Returns index where 'Java' starts

💡 Mini Challenge: Write a program that asks for a sentence and replaces one word with another.

4️⃣ Checking String Content 🔍


text = "Hello123"
print(text.isalpha())  # False (contains numbers)
print(text.isdigit())  # False (contains letters)
print("12345".isdigit())  # True
print(text.isalnum())  # True (contains only letters and numbers, no spaces or special characters)
print("Hello!".isalnum())  # False (contains a special character)

5️⃣ Splitting and Joining Strings 🔗

Splitting Strings into Lists


sentence = "Python is fun"
words = sentence.split()
print(words)  # ['Python', 'is', 'fun']

Joining Lists into a String


words = ["Python", "is", "awesome"]
sentence = " ".join(words)
print(sentence)  # Python is awesome

6️⃣ Looping Over a String 🔄

You can iterate through each character in a string using a loop.


text = "Python"
for char in text:
    print(char)

💡 Mini Challenge: Modify the loop to print the index and character side by side.

7️⃣ Counting Vowels in a String 🏆

You can count how many vowels are in a string using a loop.


text = "Hello, Python!"
vowels = "aeiouAEIOU"
vowel_count = 0
for char in text:
    if char in vowels:
        vowel_count += 1
print("Number of vowels:", vowel_count)
