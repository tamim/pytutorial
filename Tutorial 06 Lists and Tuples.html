Tutorial 6: Lists & Tuples - Organizing Data 📜

1️⃣ Why Do We Need Lists and Tuples?

In Python, lists and tuples allow us to store multiple values in a single variable. Instead of creating separate variables for each piece of data, we can store them in a structured way.

✅ Why use Lists and Tuples? - Store multiple values in one variable. - Keep data organized and manageable. - Perform operations like sorting, filtering, and iteration efficiently.

2️⃣ Lists: Your Flexible Data Container 📋

A list is an ordered, mutable (changeable) collection of elements.

Creating a List


fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = ["hello", 42, 3.14, True]

Accessing Elements


print(fruits[0])   # Output: apple
print(fruits[-1])  # Output: cherry

Modifying a List


fruits[1] = "blueberry"  # Changing "banana" to "blueberry"
print(fruits)  # Output: ["apple", "blueberry", "cherry"]

Adding and Removing Elements


fruits.append("orange")  # Add to the end
fruits.insert(1, "grape")  # Insert at index 1
print(fruits)  # ['apple', 'grape', 'blueberry', 'cherry', 'orange']
fruits.remove("cherry")  # Remove a specific item
popped_fruit = fruits.pop()  # Removes and returns the last item

3️⃣ Looping Through Lists 🔄

Using a for Loop


for fruit in fruits:
    print(fruit)

Using enumerate() to Get Index and Value


for index, fruit in enumerate(fruits):
    print(f"Index {index}: {fruit}")

Looping Using Index


for i in range(len(fruits)):
    print(fruits[i])

4️⃣ Useful List Methods 🛠️


numbers = [3, 1, 4, 1, 5, 9]
numbers.sort()  # Sort the list
print(numbers)  # [1, 1, 3, 4, 5, 9]
numbers.reverse()  # Reverse the order
print(numbers)  # [9, 5, 4, 3, 1, 1]
print(numbers.count(1))  # Count occurrences of 1
print(len(numbers))  # Get the length of the list

5️⃣ Tuples: Immutable Data Containers 🔒

A tuple is similar to a list but cannot be changed (immutable).

Creating a Tuple


days = ("Monday", "Tuesday", "Wednesday")
numbers = (1, 2, 3, 4, 5)

Accessing Elements


print(days[0])  # Output: Monday
print(days[-1]) # Output: Wednesday

Tuple Unpacking


tuple_data = ("Alice", 25, "Engineer")
name, age, profession = tuple_data
print(name)  # Output: Alice

6️⃣ Lists vs Tuples: Key Differences ⚖️

| Feature | List | Tuple | |----------|------|-------| | Mutability | ✅ Can change | ❌ Cannot change | | Performance | Slower | Faster | | Methods | More built-in methods | Fewer methods |

✅ Use lists when you need a collection that changes. ✅ Use tuples when you need a fixed collection (e.g., coordinates, database records).
