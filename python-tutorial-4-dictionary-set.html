Tutorial 2: Exploring Dictionaries and Sets

Welcome back to your Python journey! Today, we’ll learn about two magical tools: Dictionaries and Sets. These tools will help you organize data, track unique items, and solve interesting problems. Get ready for some fun coding challenges!

1. Dictionaries: Magical Key-Value Pairs

A dictionary maps keys to values. It’s like your school locker system: each locker (key) has a unique number, and inside it (value) is your stuff.

Creating and Accessing Dictionaries

  1. Creating a Dictionary

    # Mapping fruits to their prices
fruit_prices = {"apple": 2.5, "banana": 1.2, "cherry": 3.0}

  2. Creating an Empty Dictionary

    • Start with an empty dictionary if you don’t know what to add yet.
    empty_dict = {}

  3. Accessing Values

    • Use a key to get its value:
    print(fruit_prices["apple"])  # Output: 2.5

  4. Checking if a Key Exists

    • Use the in keyword to check if a key is in the dictionary.
    if "banana" in fruit_prices:
    print("Banana exists!")
else:
    print("No banana found.")

  5. What If the Key Doesn’t Exist?

    • Accessing a missing key causes an error:
    print(fruit_prices["mango"])  # KeyError: 'mango'

  6. How .get() Can Help

    • Avoid errors with .get(), which lets you specify a default value.
    print(fruit_prices.get("mango", "Not Found"))  # Output: Not Found

Adding, Updating, and Removing Items

  1. Adding or Updating Items

    fruit_prices["mango"] = 3.5  # Adds a new key-value pair
fruit_prices["apple"] = 2.8  # Updates an existing key

  2. Removing Items

    removed_price = fruit_prices.pop("banana")
print(removed_price)  # Output: 1.2
print(fruit_prices)  # Output: {'apple': 2.8, 'cherry': 3.0, 'mango': 3.5}

Iterating Over Dictionaries

  1. Looping Through Keys

    for fruit in fruit_prices:
    print(fruit)

  2. Looping Through Keys and Values

    for fruit, price in fruit_prices.items():
    print(f"The price of {fruit} is ${price}")

2. Sets: Unique and Magical Collections

Sets are like magical bags that only keep unique items. They are perfect for removing duplicates or comparing collections.

Creating and Using Sets

  1. Creating a Set

    fruits = {"apple", "banana", "cherry"}

  2. Creating an Empty Set

    • Use the set() function for an empty set. {} creates an empty dictionary.
    empty_set = set()

  3. Adding and Removing Items

    fruits.add("mango")
fruits.remove("banana")
print(fruits)  # Output: {'apple', 'cherry', 'mango'}

  4. Checking Membership

    • Use in to check if an item exists in a set.
    if "apple" in fruits:
    print("Apple is in the set!")
else:
    print("No apple found.")

  5. Iterating Over a Set

    • Loop through all items in a set:
    for fruit in fruits:
    print(fruit)

Cool Set Operations

  1. Union: Combine Two Sets

    A = {1, 2, 3}
B = {3, 4, 5}
print(A | B)  # Output: {1, 2, 3, 4, 5}
print(A.union(B))  # Output: {1, 2, 3, 4, 5}

  2. Intersection: Find Common Items

    print(A & B)  # Output: {3}
print(A.intersection(B))  # Output: {3}

  3. Difference: Items in One Set but Not the Other

    print(A - B)  # Output: {1, 2}

Project 2: Graph Generator

Objective: Store data in a dictionary and visualize it as a bar chart.

Task:

  1. Create a dictionary of countries and their populations.
  2. Use matplotlib to generate a bar chart.

Code Example:

import matplotlib.pyplot as plt

data = {"USA": 331, "India": 1391, "China": 1441, "Brazil": 213}
countries = list(data.keys())
populations = list(data.values())

plt.bar(countries, populations)
plt.xlabel("Countries")
plt.ylabel("Population (millions)")
plt.title("Country Populations")
plt.show()

Project 3: Election Tracker

Imagine your school is holding an election for a class captain. Each student gets one vote. Your task is to track the voters to ensure no one votes more than once.

  1. Use a set to store the IDs of students who have already voted.
  2. If a student tries to vote twice, notify them that their vote is invalid.

Example Input:

Voter IDs: [101, 102, 103, 101, 104]

Example Output:

101 has already voted!
Valid Votes: {101, 102, 103, 104}

Code Starter:

voters = [101, 102, 103, 101, 104]
unique_voters = set()

for voter in voters:
    if voter in unique_voters:
        print(f"{voter} has already voted!")
    else:
        unique_voters.add(voter)

print("Valid Votes:", unique_voters)
